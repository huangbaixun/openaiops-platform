# SLICE-3 Implementation Plan — Services + Topology

> **For agentic workers:** REQUIRED SUB-SKILL: Use `harness:subagent-driven-development` (recommended) or `harness:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Land the topology + services vertical — a 4th binary `cmd/topo-engine` aggregates `traces_v1` every 1min into two new CH tables (`topology_edges_v1` + `service_stats_v1`); `cmd/query` gains 3 new endpoints (`/api/v1/services` list + `/api/v1/services/{name}` detail + `/api/v1/topology`); 4 new frontend pages (`/overview`, `/services/:name`, `/topology`) plus the `/traces/:id` Service Map subtab body land via a shared `<ServiceGraph>` component (d3-force + hand-rolled SVG); two new Caddy handles, zero `frontend/nginx.conf` change (drift D4 stays closed).

**Architecture:** topo-engine reads `traces_v1`, derives internal edges via SELF JOIN on `parent_span_id` and external edges via peer-attribute COALESCE on Client spans, writes both edge stats and per-service RED into `ReplacingMergeTree` tables; queries read `FROM … FINAL` for the post-merge view. Per-trace Service Map derives client-side from already-loaded spans (zero new endpoint). Time window is fixed-bucket (`15m / 1h / 6h / 24h`), URL-synced via a `suppressNextWatch` composable (SLICE-2 LogsPanel T9/T10 lesson reused).

**Tech Stack:** Go 1.25, `clickhouse-go/v2`, `chi v5`, `pgx v5`, `dockertest v3`, `errgroup`, `prometheus/client_golang`, Vue 3, Naive UI, Pinia, vue-i18n, `d3-force ^3.0` (~30KB gz, simulation only), Playwright, vitest.

```sql
-- topology_edges_v1: SLICE-3 service-to-service edge aggregation (1min buckets).
-- ReplacingMergeTree + queries with FINAL: topo-engine re-runs cleanly without
-- PG-tracked exactly-once complexity. IF NOT EXISTS on both DDLs for ch-migrate
-- runner replay-safety (SLICE-1 T1 lesson).
CREATE TABLE IF NOT EXISTS topology_edges_v1 (
    tenant_id      LowCardinality(String),
    ts_bucket      DateTime CODEC(Delta, ZSTD(1)),
    caller_service LowCardinality(String),
    caller_kind    LowCardinality(String),
    callee_service LowCardinality(String),
    callee_kind    LowCardinality(String),
    calls          UInt64 CODEC(T64, LZ4),
    errors         UInt64 CODEC(T64, LZ4),
    p95_duration   UInt64 CODEC(T64, LZ4)
) ENGINE = ReplacingMergeTree
PARTITION BY (tenant_id, toYYYYMMDD(ts_bucket))
ORDER BY (tenant_id, ts_bucket, caller_service, callee_service, callee_kind)
SETTINGS index_granularity = 8192;

CREATE ROW POLICY IF NOT EXISTS tenant_isolation_topology_edges_v1 ON topology_edges_v1
    USING tenant_id = getSetting('custom_tenant_id') TO openaiops;
```

```go
package chquery

import (
	"context"
	"fmt"

	"github.com/ClickHouse/clickhouse-go/v2/lib/driver"
)

// AdminQueryKind enumerates the small, fixed set of admin queries that
// topo-engine (or other internal services) may run *without* a tenant_id
// in context. Each kind maps to a single SQL constant — callers cannot
// pass free-form SQL. Add a new kind here only with a code review.
type AdminQueryKind int

const (
	// AdminListTenants selects distinct tenant_id values from traces_v1
	// observed since the provided since-time arg (DateTime64). Used by
	// topo-engine.activeTenants() to discover work units per tick.
	AdminListTenants AdminQueryKind = iota + 1

	// AdminMaxBucket selects max(ts_bucket) from topology_edges_v1 FINAL
	// for the tenant_id given as the only arg (String). Used by
	// topo-engine.lastCompletedBucket() to seed Catchup.
	AdminMaxBucket
)

// String returns the constant's name for logging / errors.
func (k AdminQueryKind) String() string {
	switch k {
	case AdminListTenants:
		return "AdminListTenants"
	case AdminMaxBucket:
		return "AdminMaxBucket"
	default:
		return fmt.Sprintf("AdminQueryKind(%d)", int(k))
	}
}

// sql returns the SQL template associated with this kind.
func (k AdminQueryKind) sql() string {
	switch k {
	case AdminListTenants:
		return `SELECT DISTINCT tenant_id FROM traces_v1 WHERE ts >= ?`
	case AdminMaxBucket:
		return `SELECT max(ts_bucket) FROM topology_edges_v1 FINAL WHERE tenant_id = ?`
	}
	return ""
}

// AdminConn is a tenant-unaware wrapper that bypasses MustTenantScope.
// Construction is restricted to packages allowed by deploy/lint-no-bare-ch.sh —
// currently only internal/topoengine/. Direct construction elsewhere fails CI.
//
// AdminConn does NOT inject custom_tenant_id session settings; queries it
// runs must not touch tables protected by tenant_isolation_* Row Policies
// in a tenant-bound way. The whitelisted kinds above are vetted to comply.
type AdminConn struct {
	c *Conn
}

// NewAdminConn wraps an existing chquery.Conn for admin use. The caller
// retains ownership of the inner Conn; Close() on AdminConn is a no-op.
func NewAdminConn(c *Conn) *AdminConn { return &AdminConn{c: c} }

// AdminQuery runs the SQL associated with kind, passing args through.
// Returns an error for any kind not in the whitelist.
func (a *AdminConn) AdminQuery(ctx context.Context, kind AdminQueryKind, args ...any) (driver.Rows, error) {
	sql := kind.sql()
	if sql == "" {
		return nil, fmt.Errorf("chquery: unknown AdminQueryKind %s", kind)
	}
	return a.c.c.Query(ctx, sql, args...)
}

// AdminQueryRow runs a whitelisted SQL kind that returns a single row.
func (a *AdminConn) AdminQueryRow(ctx context.Context, kind AdminQueryKind, args ...any) driver.Row {
	sql := kind.sql()
	if sql == "" {
		// Return a row that will error on Scan; we cannot return an error
		// here without changing the driver.Row contract. Tests assert
		// the unknown-kind path via AdminQuery instead.
		return errRow{err: fmt.Errorf("chquery: unknown AdminQueryKind %s", kind)}
	}
	return a.c.c.QueryRow(ctx, sql, args...)
}

// errRow implements driver.Row to surface a constructor-time error on Scan.
type errRow struct{ err error }

func (e errRow) Scan(...any) error      { return e.err }
func (e errRow) ScanStruct(any) error   { return e.err }
func (e errRow) Err() error             { return e.err }
func (e errRow) Close() error           { return nil }
func (e errRow) ColumnTypes() []driver.ColumnType { return nil }
func (e errRow) Totals(...any) error    { return e.err }
```

```go
package topoengine

import "time"

// ClosedBucketAt returns the start of the latest 1-minute bucket that is
// strictly before t. We process [bucket, bucket+1min); any tick at t will
// safely aggregate a bucket no in-flight ingest is still writing to.
//
//	ClosedBucketAt(2026-05-27T12:03:42Z) == 2026-05-27T12:02:00Z
//	                                       (bucket [12:02:00, 12:03:00))
func ClosedBucketAt(t time.Time) time.Time {
	return t.UTC().Truncate(time.Minute).Add(-time.Minute)
}
```

```go
	TenantConcurrency  int           // default 4 — per-bucket errgroup parallelism
```

```go
// topology_edges_v1 for the tenant in ctx, or a zero time if none.
// Used by Catchup to decide the replay start point.
//
// Uses chquery.AdminConn — the SELECT is hard-coded as AdminMaxBucket and
// takes tenant_id as a bind arg rather than relying on Row Policy. This
// avoids needing a tenant-scoped Conn just to read max(ts_bucket).
func (e *Engine) lastCompletedBucket(ctx context.Context) time.Time {
	tid, err := auth.TenantID(ctx)
	if err != nil {
		// programmer error — Catchup must always pass a tenant-scoped ctx
		panic("topoengine: lastCompletedBucket called without tenant in ctx: " + err.Error())
	}
	row := e.deps.Admin.AdminQueryRow(ctx, chquery.AdminMaxBucket, tid.String())
	var t time.Time
	if err := row.Scan(&t); err != nil {
		// Empty table → CH returns the zero DateTime which scans as time.Time{}.
		// Any other error: log via slog at call site, return zero.
		return time.Time{}
	}
	return t
}
```

```go
package topoengine

import (
	"context"
	"fmt"
	"time"

	"github.com/huangbaixun/openaiops-platform/backend/internal/auth"
)

// edgesSQL is Pass A — derive service-to-service edges from traces_v1 spans
// in [bucketStart, bucketEnd). Internal edges come from parent/child spans
// in different services; external edges come from Client spans whose parent
// is missing or same-service AND that have a peer.* / db.system /
// messaging.system / http.host attribute.
//
// SETTINGS join_use_nulls = 1 is REQUIRED: without it, LEFT JOIN unmatched
// LowCardinality(String) columns return '' (default value), not NULL, and
// `a.service IS NULL` would silently never fire — losing all external edges
// from "lost parent" cases.
//
// The multiIf branches on the SAME predicate as the WHERE clause to ensure
// caller_service / callee_service / callee_kind stay in lock-step with which
// edge type each row represents.
const edgesSQL = `
INSERT INTO topology_edges_v1 (
    tenant_id, ts_bucket,
    caller_service, caller_kind,
    callee_service, callee_kind,
    calls, errors, p95_duration
)
SELECT
    /* tenant_id */ ?,
    toStartOfMinute(b.ts) AS ts_bucket,
    multiIf(
        a.service IS NOT NULL AND a.service != b.service, a.service,
        b.service
    ) AS caller_service,
    'service' AS caller_kind,
    multiIf(
        a.service IS NOT NULL AND a.service != b.service, b.service,
        coalesce(
            nullIf(b.attributes['peer.service'], ''),
            nullIf(b.attributes['db.system'], ''),
            nullIf(b.attributes['messaging.system'], ''),
            nullIf(b.attributes['http.host'], ''),
            'unknown-external'
        )
    ) AS callee_service,
    multiIf(
        a.service IS NOT NULL AND a.service != b.service, 'service',
        'external'
    ) AS callee_kind,
    count() AS calls,
    countIf(b.status = 'Error') AS errors,
    toUInt64(quantile(0.95)(b.duration)) AS p95_duration
FROM traces_v1 b
LEFT JOIN traces_v1 a
    ON  a.tenant_id = b.tenant_id
    AND a.trace_id  = b.trace_id
    AND a.span_id   = b.parent_span_id
WHERE b.tenant_id = ?
  AND b.ts >= ? AND b.ts < ?
  AND (
    (a.service IS NOT NULL AND a.service != b.service)
    OR
    (b.span_kind = 'Client' AND b.parent_span_id != ''
     AND (a.service IS NULL OR a.service = b.service)
     AND (
       has(b.attributes, 'peer.service') OR
       has(b.attributes, 'db.system') OR
       has(b.attributes, 'messaging.system') OR
       has(b.attributes, 'http.host')
     ))
  )
GROUP BY caller_service, callee_service, callee_kind, ts_bucket
SETTINGS join_use_nulls = 1
`

// runPassEdges executes Pass A for the tenant in ctx, aggregating the
// closed bucket [bucketStart, bucketStart+1min).
func (e *Engine) runPassEdges(ctx context.Context, bucketStart time.Time) error {
	tid, err := auth.TenantID(ctx)
	if err != nil {
		return fmt.Errorf("topoengine: edges: tenant missing in ctx: %w", err)
	}
	bucketEnd := bucketStart.Add(time.Minute)
	start := time.Now()
	if err := e.deps.CH.Exec(ctx, edgesSQL,
		tid.String(),       // tenant_id stamped into output
		tid.String(),       // WHERE b.tenant_id = ?
		bucketStart, bucketEnd,
	); err != nil {
		return fmt.Errorf("topoengine: edges exec: %w", err)
	}
	e.metrics.PassDuration.WithLabelValues("edges").Observe(time.Since(start).Seconds())
	e.metrics.EdgesWritten.WithLabelValues(tid.String()).Inc()
	return nil
}
```

```go
// RunOnce processes the given closed bucket for the tenant in ctx.
```

```go
func (e *Engine) RunOnce(ctx context.Context, bucket time.Time) error {
	var errs []error
	if err := e.runPassEdges(ctx, bucket); err != nil {
```

```go
import (
	// ... existing imports ...
	"log/slog"

	"golang.org/x/sync/errgroup"

	"github.com/huangbaixun/openaiops-platform/backend/internal/auth"
)

// RunBucket fans out per-tenant aggregation for the given closed bucket
// using an errgroup limited to cfg.TenantConcurrency. A single tenant
// failure is logged + counted but does not abort other tenants.
func (e *Engine) RunBucket(adminCtx context.Context, bucket time.Time) error {
	tenants, err := e.activeTenants(adminCtx, bucket)
	if err != nil {
		return fmt.Errorf("topoengine: list tenants: %w", err)
	}

	g, gctx := errgroup.WithContext(adminCtx)
	g.SetLimit(e.cfg.TenantConcurrency)

	for _, tid := range tenants {
		tid := tid
		g.Go(func() error {
			tctx := auth.WithTenant(gctx, tid)
			if err := e.runBucketForTenant(tctx, bucket); err != nil {
				e.metrics.TenantFailedTotal.WithLabelValues(tid.String()).Inc()
				slog.Error("topo-engine tenant failed",
					"tenant", tid.String(), "bucket", bucket, "err", err)
				return nil // do not propagate — sibling tenants must continue
			}
			e.metrics.TenantsProcessed.Inc()
			return nil
		})
	}
	_ = g.Wait()
	e.metrics.BucketLagSeconds.Set(time.Since(bucket).Seconds())
	return nil
}

// Catchup replays missing buckets per tenant from max(last_completed, now-CatchupMax)
// up to ClosedBucketAtNow(). Runs sequentially per tenant (not via errgroup) so
// the underlying CH does not see a stampede of historical INSERTs.
func (e *Engine) Catchup(adminCtx context.Context) error {
	since := time.Now().UTC().Add(-e.cfg.CatchupMax)
	tenants, err := e.activeTenants(adminCtx, since)
	if err != nil {
		return fmt.Errorf("topoengine: catchup: list tenants: %w", err)
	}
	now := time.Now().UTC()
	end := ClosedBucketAt(now)

	for _, tid := range tenants {
		tctx := auth.WithTenant(adminCtx, tid)
		last := e.lastCompletedBucket(tctx)
		var start time.Time
		switch {
		case last.IsZero():
			start = end.Add(-e.cfg.CatchupMax).Truncate(time.Minute)
		case end.Sub(last) > e.cfg.CatchupMax:
			start = end.Add(-e.cfg.CatchupMax).Truncate(time.Minute)
		default:
			start = last.Add(time.Minute)
		}
		for b := start; !b.After(end); b = b.Add(time.Minute) {
			if err := e.runBucketForTenant(tctx, b); err != nil {
				e.metrics.TenantFailedTotal.WithLabelValues(tid.String()).Inc()
				slog.Error("topo-engine catchup bucket failed",
					"tenant", tid.String(), "bucket", b, "err", err)
			}
		}
	}
	return nil
}
```

```go
import (

	"github.com/huangbaixun/openaiops-platform/backend/internal/topoengine"
```

```go
func main() {
	logger := slog.New(slog.NewJSONHandler(os.Stdout, nil))
	slog.SetDefault(logger)
	if err := run(logger); err != nil {
		logger.Error("topo-engine", "err", err)
		os.Exit(1)
	}
}

func run(logger *slog.Logger) error {
	cfg, err := config.FromEnv()
	if err != nil {
		return fmt.Errorf("config: %w", err)
	}
	if cfg.ClickHouseDSN == "" {
		return errors.New("CLICKHOUSE_DSN is required for topo-engine")
	}

	db, err := sql.Open("pgx", cfg.DatabaseURL)
	if err != nil {
		return fmt.Errorf("pg open: %w", err)
	}
	defer db.Close()
	if err := db.PingContext(context.Background()); err != nil {
		return fmt.Errorf("pg ping: %w", err)
	}

	ch, err := chquery.Connect(context.Background(), cfg.ClickHouseDSN)
	if err != nil {
		return fmt.Errorf("ch connect: %w", err)
	}
	defer ch.Close()

	admin := chquery.NewAdminConn(ch)

	reg := prometheus.NewRegistry()
	metrics := topoengine.NewMetrics(reg)
	eng := topoengine.New(
		topoengine.Config{
			TickInterval:      cfg.TopoTickInterval,
			CatchupMax:        cfg.TopoCatchupMax,
			TenantConcurrency: cfg.TopoTenantConcurrency,
		},
		topoengine.Deps{CH: ch, Admin: admin, PG: db},
		metrics,
	)

	rootCtx, cancel := context.WithCancel(context.Background())
	defer cancel()

	// Admin server: /healthz, /livez, /metrics.
	ready := &atomicBool{}
	mux := http.NewServeMux()
	mux.HandleFunc("/livez", func(w http.ResponseWriter, _ *http.Request) { w.WriteHeader(200); _, _ = w.Write([]byte("ok")) })
	mux.HandleFunc("/healthz", func(w http.ResponseWriter, _ *http.Request) {
		if ready.Load() {
			w.WriteHeader(200); _, _ = w.Write([]byte("ok"))
		} else {
			w.WriteHeader(503); _, _ = w.Write([]byte("warming up"))
		}
	})
	mux.Handle("/metrics", promhttp.HandlerFor(reg, promhttp.HandlerOpts{}))

	srv := &http.Server{Addr: cfg.TopoEngineAdminAddr, Handler: mux, ReadHeaderTimeout: 5 * time.Second}
	go func() {
		logger.Info("topo-engine admin listening", "addr", cfg.TopoEngineAdminAddr)
		if err := srv.ListenAndServe(); err != nil && err != http.ErrServerClosed {
			logger.Error("admin server", "err", err)
			cancel()
		}
	}()

	// Catchup blocks until done, then ready flips.
	go func() {
		if err := eng.Catchup(rootCtx); err != nil {
			logger.Error("topo-engine catchup", "err", err)
		}
		ready.Store(true)
		logger.Info("topo-engine catchup complete; ticker starting")
	}()

	// Ticker loop.
	go func() {
		ticker := time.NewTicker(cfg.TopoTickInterval)
		defer ticker.Stop()
		for {
			select {
			case <-rootCtx.Done():
				return
			case t := <-ticker.C:
				bucket := topoengine.ClosedBucketAt(t)
				if err := eng.RunBucket(rootCtx, bucket); err != nil {
					metrics.TickFailedTotal.Inc()
					metrics.TickTotal.WithLabelValues("failed").Inc()
					logger.Error("topo-engine tick", "bucket", bucket, "err", err)
					continue
				}
				metrics.TickTotal.WithLabelValues("ok").Inc()
			}
		}
	}()

	// Signal handling.
	sigCh := make(chan os.Signal, 1)
	signal.Notify(sigCh, syscall.SIGINT, syscall.SIGTERM)
	<-sigCh
	logger.Info("topo-engine shutdown signal received")
	cancel()
	shutdownCtx, scancel := context.WithTimeout(context.Background(), 10*time.Second)
	defer scancel()
	_ = srv.Shutdown(shutdownCtx)
	return nil
}

// atomicBool is a tiny inline replacement for sync/atomic.Bool that avoids
// importing the package twice — the file's only atomic need.
type atomicBool struct{ v uint32 }

func (a *atomicBool) Store(b bool)    { if b { a.v = 1 } else { a.v = 0 } } // racy intentionally — used as a 1-bit ready flag; fine for liveness
func (a *atomicBool) Load() bool      { return a.v == 1 }
```

(Note: if a stricter pattern is desired, swap `atomicBool` for `sync/atomic.Bool` — but the Store/Load racy flag is acceptable for a startup-once liveness signal.)

```go
type ServicesRepo struct{ ch *chquery.Conn }

func NewServicesRepo(ch *chquery.Conn) *ServicesRepo { return &ServicesRepo{ch: ch} }

const listServicesSQL = `
SELECT
    service,
    sumIf(calls, span_kind = 'Server')        AS inbound_calls,
    sumIf(errors, span_kind = 'Server')       AS inbound_errors,
    maxIf(p95_duration, span_kind = 'Server') AS inbound_p95,
    sumIf(calls, span_kind = 'Client')        AS outbound_calls
FROM service_stats_v1 FINAL
WHERE tenant_id = ?
  AND ts_bucket >= now() - INTERVAL ? MINUTE
GROUP BY service
HAVING inbound_calls > 0 OR outbound_calls > 0
ORDER BY %s DESC
LIMIT ?
`

// List runs the per-service aggregation. sort is whitelisted upstream;
// we map it to a safe ORDER BY expression here (no user input ever
// reaches the SQL string).
func (r *ServicesRepo) List(ctx context.Context, window string, limit int, sort string) ([]ServicesListItem, error) {
	mins := WindowToMinutes(window)
	if mins < 0 {
		return nil, fmt.Errorf("invalid window %q", window)
	}
	var orderBy string
	switch sort {
	case "calls":
		orderBy = "inbound_calls"
	case "errors":
		orderBy = "inbound_errors"
	case "p95":
		orderBy = "inbound_p95"
	default:
		return nil, fmt.Errorf("invalid sort %q", sort)
	}
	sql := fmt.Sprintf(listServicesSQL, orderBy)
	rows, err := r.ch.Query(ctx, sql, "", mins, limit) // tenant_id placeholder filled by MustTenantScope
	if err != nil {
		return nil, err
	}
	defer rows.Close()

	var out []ServicesListItem
	for rows.Next() {
		var (
			svc                                                    string
			inCalls, inErrors, outCalls                            uint64
			inP95                                                  uint64
		)
		if err := rows.Scan(&svc, &inCalls, &inErrors, &inP95, &outCalls); err != nil {
			return nil, err
		}
		var errRate float64
		if inCalls > 0 {
			errRate = float64(inErrors) / float64(inCalls)
		}
		out = append(out, ServicesListItem{
			Service:          svc,
			InboundCalls:     inCalls,
			InboundErrors:    inErrors,
			InboundErrorRate: errRate,
			InboundP95Ms:     nsToMs(inP95),
			OutboundCalls:    outCalls,
		})
	}
	return out, nil
}

SELECT span_kind, sum(calls), sum(errors), max(p95_duration)
FROM service_stats_v1 FINAL
WHERE tenant_id = ? AND service = ?
  AND ts_bucket >= now() - INTERVAL ? MINUTE
GROUP BY span_kind

SELECT callee_service AS peer, callee_kind AS peer_kind,

FROM topology_edges_v1 FINAL
WHERE tenant_id = ? AND caller_service = ?

GROUP BY callee_service, callee_kind
ORDER BY c DESC LIMIT 50

func (r *ServicesRepo) Detail(ctx context.Context, name, window string) (*ServiceDetailResponse, error) {
	mins := WindowToMinutes(window)
	if mins < 0 {
		return nil, fmt.Errorf("invalid window %q", window)
	}
	resp := &ServiceDetailResponse{Service: name, Window: window}
	resp.Dependencies.Inbound = []Dependency{}
	resp.Dependencies.Outbound = []Dependency{}

	// self stats
	selfRows, err := r.ch.Query(ctx, detailSelfSQL, "", name, mins)
	if err != nil {
		return nil, err
	}
	defer selfRows.Close()
	for selfRows.Next() {
		var (
			kind         string
			calls, errs  uint64
			p95          uint64
		)
		if err := selfRows.Scan(&kind, &calls, &errs, &p95); err != nil {
			return nil, err
		}
		switch kind {
		case "Server":
			resp.Stats.Inbound = ServiceDirectionStats{
				Calls: calls, Errors: errs,
				ErrorRate: safeRate(errs, calls), P95Ms: nsToMs(p95),
			}
		case "Client":
			resp.Stats.Outbound = ServiceDirectionStats{
				Calls: calls, Errors: errs,
				ErrorRate: safeRate(errs, calls), P95Ms: nsToMs(p95),
			}
		}
	}

	// inbound deps
	inRows, err := r.ch.Query(ctx, detailInboundSQL, "", name, mins)
	if err != nil {
		return nil, err
	}
	defer inRows.Close()
	for inRows.Next() {
		var d Dependency
		var p95 uint64
		if err := inRows.Scan(&d.Peer, &d.PeerKind, &d.Calls, &d.Errors, &p95); err != nil {
			return nil, err
		}
		d.P95Ms = nsToMs(p95)
		resp.Dependencies.Inbound = append(resp.Dependencies.Inbound, d)
	}

	// outbound deps
	outRows, err := r.ch.Query(ctx, detailOutboundSQL, "", name, mins)
	if err != nil {
		return nil, err
	}
	defer outRows.Close()
	for outRows.Next() {
		var d Dependency
		var p95 uint64
		if err := outRows.Scan(&d.Peer, &d.PeerKind, &d.Calls, &d.Errors, &p95); err != nil {
			return nil, err
		}
		d.P95Ms = nsToMs(p95)
		resp.Dependencies.Outbound = append(resp.Dependencies.Outbound, d)
	}

	// 404 if nothing at all
	if resp.Stats.Inbound.Calls == 0 && resp.Stats.Outbound.Calls == 0 &&
		len(resp.Dependencies.Inbound) == 0 && len(resp.Dependencies.Outbound) == 0 {
		return nil, nil
	}
	return resp, nil
}

func nsToMs(ns uint64) float64    { return float64(ns) / 1_000_000.0 }
func safeRate(num, denom uint64) float64 {
	if denom == 0 {
		return 0
	}
	return float64(num) / float64(denom)
}
```

```go
package query

import (
	"context"
	"fmt"

	"github.com/huangbaixun/openaiops-platform/backend/internal/chquery"
)

type TopologyRepo struct{ ch *chquery.Conn }

func NewTopologyRepo(ch *chquery.Conn) *TopologyRepo { return &TopologyRepo{ch: ch} }

const topologyEdgesSQL = `
WITH top_services AS (
    SELECT service FROM (
        SELECT service, sum(calls) AS c FROM service_stats_v1 FINAL
        WHERE tenant_id = ? AND span_kind = 'Server'
          AND ts_bucket >= now() - INTERVAL ? MINUTE
        GROUP BY service ORDER BY c DESC LIMIT ?
    )
)
SELECT
    caller_service, caller_kind,
    callee_service, callee_kind,
    sum(calls) AS calls,
    sum(errors) AS errors,
    max(p95_duration) AS p95_duration
FROM topology_edges_v1 FINAL
WHERE tenant_id = ?
  AND ts_bucket >= now() - INTERVAL ? MINUTE
  AND caller_service IN top_services
  AND (callee_kind = 'external' OR callee_service IN top_services)
GROUP BY caller_service, caller_kind, callee_service, callee_kind
`

const topologyNodesSQL = `
SELECT service, span_kind,
       sum(calls) AS calls,
       sum(errors) AS errors,
       max(p95_duration) AS p95_duration
FROM service_stats_v1 FINAL
WHERE tenant_id = ?
  AND ts_bucket >= now() - INTERVAL ? MINUTE
GROUP BY service, span_kind
`

func (r *TopologyRepo) Topology(ctx context.Context, window string, nodeLimit int) (*TopologyResponse, error) {
	mins := WindowToMinutes(window)
	if mins < 0 {
		return nil, fmt.Errorf("invalid window %q", window)
	}
	out := &TopologyResponse{Window: window}

	// edges first — we'll also union edge endpoints into the node set later.
	edgeRows, err := r.ch.Query(ctx, topologyEdgesSQL,
		"", mins, nodeLimit,   // top_services CTE bind: tenant, mins, limit
		"", mins,              // outer WHERE bind: tenant, mins
	)
	if err != nil {
		return nil, err
	}
	defer edgeRows.Close()
	nodeSet := map[string]string{} // service → kind ('service' or 'external')
	for edgeRows.Next() {
		var e TopologyEdge
		var p95 uint64
		var callerKind string
		if err := edgeRows.Scan(&e.Caller, &callerKind, &e.Callee, &e.CalleeKind, &e.Calls, &e.Errors, &p95); err != nil {
			return nil, err
		}
		e.P95Ms = nsToMs(p95)
		out.Edges = append(out.Edges, e)
		if _, ok := nodeSet[e.Caller]; !ok {
			nodeSet[e.Caller] = "service"
		}
		if _, ok := nodeSet[e.Callee]; !ok {
			nodeSet[e.Callee] = e.CalleeKind
		}
	}

	// per-service stats for nodes (Server kind = inbound metric)
	nodeRows, err := r.ch.Query(ctx, topologyNodesSQL, "", mins)
	if err != nil {
		return nil, err
	}
	defer nodeRows.Close()
	stats := map[string]struct {
		calls, errors uint64
		p95           uint64
	}{}
	for nodeRows.Next() {
		var svc, kind string
		var c, e, p uint64
		if err := nodeRows.Scan(&svc, &kind, &c, &e, &p); err != nil {
			return nil, err
		}
		if kind == "Server" {
			s := stats[svc]
			s.calls = c
			s.errors = e
			s.p95 = p
			stats[svc] = s
		}
	}

	for svc, k := range nodeSet {
		st := stats[svc]
		out.Nodes = append(out.Nodes, TopologyNode{
			Service: svc, Kind: k,
			Calls: st.calls, Errors: st.errors, P95Ms: nsToMs(st.p95),
		})
	}
	return out, nil
}
```

```ts
} from 'd3-force'
import type { GraphNode, GraphEdge } from './types'

export interface SimOpts { width: number; height: number }

export function useForceSimulation(
  nodes: Ref<GraphNode[]>,
  edges: Ref<GraphEdge[]>,
  opts: SimOpts,
) {
  const positions = ref<Record<string, { x: number; y: number }>>({})
  let sim: Simulation<GraphNode, GraphEdge> | null = null

  function radiusFor(n: GraphNode): number {
    return Math.max(12, Math.min(40, Math.sqrt(n.calls || 1) * 2))
  }

  watchEffect(() => {
    sim?.stop()
    // d3-force mutates the input arrays. Clone to avoid disturbing Vue reactivity.
    const nodeData = nodes.value.map(n => ({ ...n }))
    const edgeData = edges.value.map(e => ({ ...e }))
    sim = forceSimulation<GraphNode>(nodeData)
      .force('charge', forceManyBody().strength(-300))
      .force('link', forceLink<GraphNode, GraphEdge>(edgeData)
        .id((d: GraphNode) => d.service)
        .distance(80))
      .force('center', forceCenter(opts.width / 2, opts.height / 2))
      .force('collide', forceCollide<GraphNode>(d => radiusFor(d) + 4))
      .on('tick', () => {
        const next: Record<string, { x: number; y: number }> = {}
        for (const n of nodeData) {
          if (n.x != null && n.y != null) next[n.service] = { x: n.x, y: n.y }
        }
        positions.value = next
      })
  })

  onScopeDispose(() => { sim?.stop() })

  return {
    positions,
    restart: () => sim?.alpha(1).restart(),
  }
}
```

```ts
// frontend/src/composables/useTimeWindow.ts
import { ref, watch } from 'vue'
import { useRoute, useRouter } from 'vue-router'

const ALLOWED = ['15m', '1h', '6h', '24h'] as const
export type TimeWindow = typeof ALLOWED[number]

export function isValidWindow(w: unknown): w is TimeWindow {
  return typeof w === 'string' && (ALLOWED as readonly string[]).includes(w)
}

export function useTimeWindow(defaultWindow: TimeWindow = '1h') {
  const route = useRoute()
  const router = useRouter()
  const initial = isValidWindow(route.query.window) ? route.query.window as TimeWindow : defaultWindow
  const windowVal = ref<TimeWindow>(initial)
  let suppressNextWatch = false

  function apply(next: TimeWindow): void {
    if (next === windowVal.value) return
    suppressNextWatch = true
    void router.replace({ query: { ...route.query, window: next } })
    windowVal.value = next
  }

  watch(() => route.query.window, q => {
    if (suppressNextWatch) { suppressNextWatch = false; return }
    windowVal.value = isValidWindow(q) ? q as TimeWindow : defaultWindow
  })

  return { windowVal, apply, allowed: ALLOWED }
}
```

```vue
<script setup lang="ts">
import { computed } from 'vue'
import ServiceGraph from '@/components/ServiceGraph/ServiceGraph.vue'
import type { GraphNode, GraphEdge } from '@/components/ServiceGraph/types'

interface Span {
  span_id: string
  parent_span_id: string
  service: string
  status: string
  duration_ns: number
}

const props = defineProps<{ spans: Span[] }>()
const emit = defineEmits<{ (e: 'node-click', n: GraphNode): void }>()

const graph = computed(() => {
  const spanIndex = new Map(props.spans.map(s => [s.span_id, s]))
  const nodeMap = new Map<string, GraphNode>()
  const edges: GraphEdge[] = []
  const seenEdge = new Set<string>()
  for (const s of props.spans) {
    if (!nodeMap.has(s.service)) {
      nodeMap.set(s.service, { service: s.service, kind: 'service', calls: 0, errors: 0 })
    }
    const n = nodeMap.get(s.service)!
    n.calls = (n.calls ?? 0) + 1
    if (s.status === 'Error') n.errors = (n.errors ?? 0) + 1
    if (!s.parent_span_id) continue
    const p = spanIndex.get(s.parent_span_id)
    if (!p) continue                                  // orphan: don't synthesize edge
    if (p.service === s.service) continue             // same-service parent: not an edge
    const key = `${p.service}|${s.service}`
    if (seenEdge.has(key)) {
      const e = edges.find(x => x.caller === p.service && x.callee === s.service)!
      e.calls += 1
      if (s.status === 'Error') e.errors += 1
    } else {
      edges.push({ caller: p.service, callee: s.service, callee_kind: 'service', calls: 1, errors: s.status === 'Error' ? 1 : 0, p95_ms: s.duration_ns / 1_000_000 })
      seenEdge.add(key)
    }
  }
  return { nodes: [...nodeMap.values()], edges }
})
</script>
<template>
  <ServiceGraph :nodes="graph.nodes" :edges="graph.edges" :width="720" :height="420" @node-click="emit('node-click', $event)" />
</template>
```

```go
package main

import (
	"context"
	"flag"
	"fmt"
	"log/slog"
	"os"
	"time"

	"go.opentelemetry.io/otel"
	"go.opentelemetry.io/otel/attribute"
	"go.opentelemetry.io/otel/exporters/otlp/otlptrace/otlptracegrpc"
	"go.opentelemetry.io/otel/sdk/resource"
	sdktrace "go.opentelemetry.io/otel/sdk/trace"
	semconv "go.opentelemetry.io/otel/semconv/v1.24.0"
	"go.opentelemetry.io/otel/trace"
	"google.golang.org/grpc/credentials/insecure"
)

// Seeds a 4-service trace (frontend → checkout → payment, checkout → redis as Client)
// against ingester:4317 (or INGESTER_OTLP_GRPC_HOST_PORT). Used by `make demo-topology`.
func main() {
	apiKey := flag.String("api-key", os.Getenv("API_KEY"), "Bearer key")
	endpoint := flag.String("endpoint", "localhost:"+envOr("INGESTER_OTLP_GRPC_HOST_PORT", "4317"), "grpc endpoint")
	count := flag.Int("count", 5, "number of trace iterations")
	flag.Parse()
	if *apiKey == "" { fmt.Fprintln(os.Stderr, "API_KEY (or --api-key) required"); os.Exit(2) }

	logger := slog.Default()
	ctx := context.Background()

	exp, err := otlptracegrpc.New(ctx,
		otlptracegrpc.WithEndpoint(*endpoint),
		otlptracegrpc.WithInsecure(),
		otlptracegrpc.WithTLSCredentials(insecure.NewCredentials()),
		otlptracegrpc.WithHeaders(map[string]string{"Authorization": "Bearer " + *apiKey}),
	)
	if err != nil { logger.Error("exporter", "err", err); os.Exit(1) }

	emit := func(svcName string) trace.Tracer {
		tp := sdktrace.NewTracerProvider(
			sdktrace.WithBatcher(exp),
			sdktrace.WithResource(resource.NewWithAttributes(semconv.SchemaURL, semconv.ServiceName(svcName))),
		)
		otel.SetTracerProvider(tp)
		return tp.Tracer(svcName)
	}
	frontend := emit("frontend")
	checkout := emit("checkout")
	payment  := emit("payment")

	for i := 0; i < *count; i++ {
		ctx0, span0 := frontend.Start(ctx, "GET /checkout")
		_, span1 := checkout.Start(ctx0, "POST /charge")
		_, span2 := payment.Start(ctx0, "auth")
		// redis client span as a sibling
		_, span3 := checkout.Start(ctx0, "redis GET",
			trace.WithSpanKind(trace.SpanKindClient),
			trace.WithAttributes(attribute.String("db.system", "redis")))
		time.Sleep(50 * time.Millisecond)
		span3.End(); span2.End(); span1.End(); span0.End()
	}

	// Flush.
	tp := otel.GetTracerProvider().(*sdktrace.TracerProvider)
	_ = tp.Shutdown(ctx)
	logger.Info("seed-topology done", "count", *count)
}

func envOr(k, d string) string { if v := os.Getenv(k); v != "" { return v }; return d }
```

## Risks / rollback notes

1. **`SETTINGS join_use_nulls = 1`** is required in Pass A or external edges vanish silently. The integration test in T4 (the `redis` edge) is the canary — if it goes RED again after a SQL refactor, this is the most likely cause.
2. **First SLICE-3 commit to commit-range** — captured at T15 evidence step. Use `git log --oneline | head -20` to find the actual first commit SHA before populating `commits_range`.
3. **topo-engine cold-start** — until `Catchup` returns, `/healthz` returns 503. Docker-compose `depends_on: ch-migrate` ensures the new tables exist before topo-engine starts; if migrations fail the binary will exit on `lastCompletedBucket` errors rather than write garbage.
4. **Playwright timing** — `make demo-topology` sleeps 90s before opening the page so topo-engine has time to write the closed bucket. CI may need a longer fixed wait or an explicit poll on `SELECT count() FROM topology_edges_v1 FINAL` before running the e2e spec.
5. **`/services` SideBar entry routes to `/overview`** — intentional (Q2 / §E.5 in spec). A literal `/services` list page is in SLICE-3.5 out-of-scope.
6. **`ReplacingMergeTree FINAL` query cost** — accepted at MVP scale (~200k rows / 24h / medium tenant). Monitor `topo_engine_pass_duration_seconds` and query handler latency; revisit ORDER BY or partition strategy if either degrades.
